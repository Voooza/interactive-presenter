# Welcome to Interactive Presenter

# What is this?

A **Markdown-driven** presentation tool that lets audiences participate
via *emoji reactions*, polls, Q&A, and more.

# Key Features

- Load any `.md` file as a presentation
- Each `# Heading` becomes a slide
- Navigate with arrow keys or spacebar
- Audience interaction coming in v2

# Code Example

Here is how simple a presentation file is:

```markdown
# My First Slide

Hello, world!

# My Second Slide

More content here.
```
